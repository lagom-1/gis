# 登录页面设计规范

## 概述

为 OpenGIS 项目设计全新的用户登录页面，采用企业级风格 + 左右分栏布局，支持多种认证方式。

## 设计目标

- 专业的企业级视觉风格
- 清晰的视觉层次和交互反馈
- 支持多种认证方式（密码、验证码、第三方）
- 良好的响应式适配
- 与现有 UI 系统保持一致

## 设计方案

### 1. 整体布局

采用左右分栏布局：
- **左侧**：品牌展示区，绿色渐变背景，展示 OpenGIS 品牌和 GIS 特色功能
- **右侧**：登录表单区，白色背景，提供多种登录方式

### 2. 视觉风格

#### 颜色系统
- **主色调**：生态绿色 (#059669)
- **渐变**：从 #059669 到 #10b981
- **背景**：#f8fafc (浅灰)
- **文字**：#1e293b (深灰)、#64748b (中灰)、#94a3b8 (浅灰)

#### 字体
- 标题：24px，粗体
- 副标题：14px，常规
- 表单标签：13px，中等粗细
- 按钮：14px，中等粗细

#### 间距
- 页面内边距：40px
- 表单元素间距：12px
- 按钮内边距：12px 16px

### 3. 认证方式

#### 3.1 密码登录（默认）
- 邮箱输入框
- 密码输入框
- 登录按钮
- 忘记密码链接

#### 3.2 邮箱验证码登录
- 邮箱输入框
- 验证码输入框 + 发送按钮
- 登录按钮

#### 3.3 第三方登录
- 微信登录：扫码授权后自动登录

### 4. 交互状态

#### 4.1 表单状态
- **默认**：灰色边框 (#d1d5db)
- **聚焦**：绿色边框 (#059669)，标签变绿
- **错误**：红色边框 (#dc2626)，错误提示
- **禁用**：灰色背景，不可点击

#### 4.2 按钮状态
- **默认**：绿色背景 (#059669)
- **悬停**：深绿色 (#047857)
- **加载**：灰色背景，旋转图标
- **禁用**：灰色背景，不可点击

### 5. 响应式设计

#### 桌面端 (>1024px)
- 左右分栏，各占 50%
- 左侧品牌区完整展示

#### 平板端 (768px-1024px)
- 左右分栏，左侧稍窄
- 品牌区简化展示

#### 移动端 (<768px)
- 上下布局
- 品牌区简化为顶部横幅
- 表单区全宽显示

### 6. 动画效果

#### 页面加载
- 左侧品牌区从左滑入
- 右侧表单区从右滑入
- 持续时间：300ms

#### 表单交互
- 输入框聚焦时边框颜色渐变
- 按钮悬停时背景色渐变
- 持续时间：150ms

#### 加载状态
- 旋转图标动画
- 持续时间：1s 线性循环

### 7. 错误处理

#### 表单验证
- 邮箱格式验证：必须符合标准邮箱格式
- 密码长度验证：至少 6 位字符
- 验证码格式验证：6 位数字验证码

#### 服务端错误
- 网络错误提示
- 账号不存在提示
- 密码错误提示
- 验证码错误提示

### 8. 无障碍设计

- 所有表单元素有关联的 label
- 颜色对比度符合 WCAG 标准
- 键盘导航支持
- 屏幕阅读器支持

## 技术实现

### 组件结构

```
LoginPage
├── BrandPanel (左侧品牌区)
│   ├── Logo
│   ├── Title
│   ├── Description
│   └── Features
└── LoginForm (右侧表单区)
    ├── TabSwitcher (登录方式切换)
    ├── PasswordForm (密码登录表单)
    ├── CodeForm (验证码登录表单)
    ├── ThirdPartyLogin (第三方登录)
    └── Footer (注册链接等)
```

### 状态管理

使用 Zustand 管理以下状态：
- 当前登录方式（密码/验证码/第三方）
- 表单数据
- 加载状态
- 错误信息

### API 接口

- `POST /api/auth/login` - 密码登录
- `POST /api/auth/send-code` - 发送验证码
- `POST /api/auth/login-with-code` - 验证码登录
- `POST /api/auth/wechat` - 微信登录

## 与现有系统集成

### 路由
- `/login` - 登录页面
- `/register` - 注册页面（保持现有）

### 状态管理
- 使用现有的 `useAppStore` 中的 `login` 方法
- 扩展支持新的登录方式

### 样式系统
- 使用现有的 TailwindCSS 配置
- 添加新的颜色和动画类
